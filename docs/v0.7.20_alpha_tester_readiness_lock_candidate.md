
It does **not** bypass `main.py`.

## Current Supported Alpha Launch Path

Supported v0.7 alpha launch path:

```text
Launch_The_Dragons_Touch.pyw
```

Alpha testers should launch the app by opening the project folder and double-clicking this file.

Visual Studio Code is **not required** to launch the app through the `.pyw` launcher.

The following remain developer fallback / experimental only:

```text
Launch_The_Dragons_Touch.py
Launch_The_Dragons_Touch.bat
Create_Desktop_Shortcut.py
Create_Desktop_Shortcut.pyw
Create_Desktop_Shortcut.bat
```

Desktop shortcut support is deferred because Smart App Control blocked the shortcut/batch path during local testing. Do not instruct alpha testers to disable Smart App Control.

---

## Current Clean Data Policy

The clean ZIP may omit:

```text
data/scryfall_cards.json
```

because it is very large.

The clean ZIP should include:

```text
download_scryfall_data.py
```

If Scryfall card data is missing, recreate it from the project root with:

```text
python download_scryfall_data.py
```

This regenerates:

```text
data/scryfall_cards.json
```

Internet access is required for the downloader.
