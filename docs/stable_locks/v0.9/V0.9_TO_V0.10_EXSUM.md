# EXSUM — The Dragon's Touch v0.9 Stable Lock Handoff

## Project

The Dragon's Touch is a local Python-based Magic: The Gathering Commander deck review assistant. It analyzes Commander decklists, identifies strategy, cut pressure, protected cards, replacement needs, collection-only candidates, full-pool fallback examples, combo-aware context, and generates AI handoff/report output.

## Current stable checkpoint

**v0.9 Stable Lock — Replacement Candidate Engine Foundation**

v0.9 is not primarily a UI polish release. It is an engine-side checkpoint proving that replacement candidate data is structured, bounded, source-labeled, confidence-labeled, and safe enough to feed v0.10 Simple View later.

Combo-aware runs are launched in PowerShell with:

```powershell

$env:MTG_COMBO_AWARENESS_ARTIFACT = "both"
py main.py
```

Combo-aware reports preserve replacement preview, fallback preview, exact preview safety labels, and aggregate output.

## Known limitations to carry forward

- v0.9 report is still AI/dev-friendly.
- Human-readable layperson view belongs mainly to v0.10 Simple View / UI.
- Exact preview is not a full ranking engine.
- Budget is labeled but not live-price checked.
- Bracket/table-fit is cautioned, not fully socially evaluated.
- Full-pool examples are not claimed as owned.
- Automatic swaps remain disabled.
- Combo examples remain suppressed unless combo optimization is explicitly enabled.
- Archetype regression is keyword-based and broad, not a semantic classifier.
