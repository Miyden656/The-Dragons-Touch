## Recommended Development Rule

Do not patch mode visibility widget-by-widget. Build mode-specific page layouts:

```python
def build_page(window):
    if window.is_user_mode():
        return build_user_page(window)
    return build_developer_page(window)
```

This prevents stale layout bugs where a page built in Developer Mode remains visible after switching back to User Mode.
